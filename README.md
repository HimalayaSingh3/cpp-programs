## Getting Started

To get started with these programs, you need to have a C++ compiler installed on your machine. If you don't have one, you can download and install [GCC](https://gcc.gnu.org/) or use an IDE like [Visual Studio](https://visualstudio.microsoft.com/) or [Code::Blocks](http://www.codeblocks.org/).
